1. form(block)에 label for + input id 로 구성된다.


2. label태그에는 .form-label(mb추가) input태그에 .form-control cls를 주면, w-100 및 style들이 적용된다.

3. checkbox의 경우, form>label과 input에 대해 form > div.form-check > input.form-check-input + label.form-check-label형태로 사용한다

- input -> label형태로 사용하며

- input은 type="checkbox" value=""가 기본이다.

4. radio의 경우, checkbox랑 같은데, type만="radio"로 변경하고, name은 여러개의 div.form-check가 다 동일하게 생성한다.

5. chekcbox or radio는 inline으로 바로 만들 수 있다.

- 각 div.form-check에 .form-check-inline을 추가된다.

6. select는 label.form-label + select.form-select > option태그들 을 사용한다.

- default option은 selected 속성을 맨 위에 추가한다

7. select에서 여러개를 선택해야한다면, multiple 속성을 추가하면 된다.

- control 누르고 다중선택한다.

- 이 때는, default option을 제거한다

8. textarea도 input type="text"처럼, label.form-label + textrea.form-control로 사용한다

이 때, cols="" rows=""가 추가된다.

9. file은 input의 type만 "file"로 바꾼다.

10. input의 스타일은 input.form-control에 form-control-size 만 추가한다.

11. range는 input.form-range 에 type="range"로 만들면 된다.

- min, max를 정해줄 수잇고

- step으로는 step="0.5" 처럼 정해주면 된다.

12. form 태그에 class="row"를 추가하고, 각 label+input에 div.col을 씌우거나, 부모가 있다면, col만 추가한다

- 각 col-sm-6으로 하면, 가장 작을때만 한줄, 그 이후(sm)부터는 절반씩 차지한다

13. checkbox or radio의 경우, div.form-check라는 부모를 가졌으니 col만 추가한다. inline은 어차피 한줄이니, 새로운 부모를 싸서 만든다
