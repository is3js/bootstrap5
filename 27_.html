1. 일단 div> 개별이미지당 div>img.w-100으로 여러개 이미지를 올려보자.

2. 이번엔 div.carousel>div.carousel-inner에 개별이미지당 div.carousel-item>img.w-100으로 여러개 이미지를 올려보자.

- 첫번째 이미지만 carousel-item에 active cls를 추가

3. div.carousel>div.carousel-inner>div.carousel-item>img.w-100.active(첫 item)에서

- div.carousel에 data-bs-ride="carousel"을 추가하면 자동으로 넘어가기 시작한다.

- div.carousel[data-bs-ride="carousel"]>div.carousel-inner>div.carousel-item>img.w-100.active(첫 item)에서


4. interval변경 시, ride되는 캐러셀의 interval은 id를 주고 -> js로 찾아서 수정해야한다.

- wrap 여부가 순환여부다. true로 줘야됨.

The .slide class adds a CSS transition and animation effect, which makes the items slide when showing a new item. Omit this class if you do not want this effect.

        
            var myCarousel = document.querySelector('#indexCarousel')
            var carousel = new bootstrap.Carousel(myCarousel, {
              interval: 1000,
              wrap: true
            })

5. 슬라이드 화살표 추가는, carousel에 carousel-inner의 형제로 button.carousel-control-prev, next 버튼 2개를 형제로 추가하는 것이다.

- 각 button에는 data-bs-target="#indexCarousel2"의 carousel id를 공통적으로 target하고

- prev버튼에는 data-bs-slide="prev", next버튼에는 data-bs-slide="next"를 추가한다

- button이 아니라 a태그로 한다면 role="button"을 추가해야한다

6. indicator를 추가하고 싶다면, carousel 자식으로서 div.carousel-indicators를 주고, 각 button들로 data-bs-target을 캐러셀로 잡는다.

- 또한, 각 button들에는 data-bs-slide-to=""에 0부터 지정해줘야한다.

7. 각 carousel-item 마다 img외 caption을 적용하려면, img태그 형제로 div.carousel-caption.d-none.d-md-block으로

- md부터 나타나는 캡션을 만들어주면 된다. 알아서 가운데 정렬 + 흰색 글자로 지정되어 있다.
